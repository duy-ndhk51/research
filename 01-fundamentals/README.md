## Suggested Learning Path

1. Complexity Analysis (Big-O notation)
2. Data Structures (arrays → linked lists → trees → graphs → hash tables)
3. Algorithms (sorting → searching → graph algorithms → dynamic programming)
4. OOP principles (SOLID, inheritance, polymorphism, encapsulation)
5. Design Patterns (creational → structural → behavioral)
6. Functional Programming (pure functions, immutability, higher-order functions)
